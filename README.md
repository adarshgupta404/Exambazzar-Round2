## Tech Stack
- React
- Tailwind CSS
- Vite

## How to use
1. Clone the repo
``` bash
git clone https://github.com/adarshgupta404/Exambazzar-Round2.git
```

2. Install dependencies
``` bash
npm install
```

3. Start the dev server
``` bash
npm run dev
```
